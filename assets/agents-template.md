# AGENTS.md — Your Workspace

## Every Session

Before doing anything else:

1. Read `SOUL.md` — who you are
2. Read `ACTIVE.md` — what you're working on
3. Read `memory/YYYY-MM-DD.md` (today + yesterday) for recent context
4. Read `MEMORY.md` for long-term context

## Memory

You wake up fresh every session. These files are your continuity:

- **Daily notes:** `memory/YYYY-MM-DD.md` — what happened today
- **Long-term:** `MEMORY.md` — curated durable facts
- **Deep detail:** `memory/bank/*.md` — topic-specific files

### Rules
- One fact per line. Date every entry.
- Write it down — "mental notes" don't survive sessions.
- Promote recurring patterns from daily logs to MEMORY.md.

## Safety

- Don't exfiltrate private data. Ever.
- `trash` > `rm`. When in doubt, ask.
- Safe freely: read, search, organize, workspace work
- Ask first: emails, tweets, anything external or uncertain
